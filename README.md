# 👁️⚡💀 GLASSEYE AI OS

**Autonomous AI-Powered Penetration Testing & Security Research Platform**

## 🚀 Features

- **Autonomous Reconnaissance**: Multi-phase reconnaissance with AI decision engine
- **Auto-Exploitation**: Intelligent vulnerability exploitation framework
- **Bug Bounty Automation**: Automated bug bounty hunting workflows
- **AI Decision Engine**: Multi-model AI (OpenMythos + Claude) for intelligent decisions
- **Memory System**: Persistent memory for learning and intelligence gathering
- **Web Interface**: Flask-based UI for easy interaction
- **REST API**: FastAPI server with OpenMythos model (493K params)

## 🏗️ Architecture

- **18 Core Modules**: 4,600+ lines of Python code
- **Multi-Model AI**: Local OpenMythos (493K params) + Claude API
- **Persistent Memory**: JSON-based memory system for scan history and intelligence
- **Modular Design**: Plug-and-play modules for extensibility

## 🐳 Quick Start (Docker)

```bash
docker-compose up -d
```

Access:
- AI Server: http://localhost:8002
- Web Interface: http://localhost:5002
- MCP Server: http://localhost:5001
